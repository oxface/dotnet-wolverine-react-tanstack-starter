# Agent Index

This repository is a lightweight starter for .NET projects using Wolverine,
React, TanStack Query, TanStack Router, pnpm, Husky, Commitlint, and Aspire.

Start with:

- [README.md](README.md) for the repo purpose and quick start.
- [docs/README.md](docs/README.md) for project guidance.
- [docs/development.md](docs/development.md) for local workflow.
- [docs/testing.md](docs/testing.md) for verification commands.
- [server/AGENTS.md](server/AGENTS.md) before changing backend behavior.
- [web/AGENTS.md](web/AGENTS.md) before changing frontend behavior.
- [orchestration/AGENTS.md](orchestration/AGENTS.md) before changing Aspire.
- [scripts/README.md](scripts/README.md) before changing init scripts.

Keep this starter boring. Do not add persistence, auth, broker transports,
modules, generated clients, or domain behavior unless a consuming project has a
concrete need.
